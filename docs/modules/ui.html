onWidgetAction

  • onWidgetAction(callback: Function, action?: string, widgetId?: string): void
    • Defined in ui.d.ts:55

    Event listener for when a widget action occurs. This is typically when a user interacts with a widget, such as pressing a button, moving a slider, toggling a switch.

    This method may either be used as a firehose:

    gui.onWidgetAction(e => console.log('any event:', e));

    Or for a specific action for a specific widget:

    gui.onWidgetAction(resetClicked, 'click', 'resetButton')

    Parameters

    • callback: Function

      Your callback will receive the widget event

    • Optional action: string

      Specifhy the vent you are interested in. If falsy, any action will apply

    • Optional widgetId: string

      Specify the widget you are interested in. If falsy, any widget will apply

    Returns void
